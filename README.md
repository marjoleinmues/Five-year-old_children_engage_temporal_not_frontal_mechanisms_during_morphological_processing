# Phonological_Semantic_Mechanisms_During_Morphological_Processing_5yo-

This is the laboratory log, participant_id and code used for the study entitled "Examining the role of phonological and semantic mechanisms during morphological processing of sentences in five-year-old children/Five-year-old children engage temporal but not frontal mechanisms during morphological processing of sentences.

The data is available open access via https://openneuro.org/datasets/ds003604/versions/1.0.7 The analytical plan was preregistered here: https://osf.io/juzcm
